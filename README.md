# 项目简介 java137

本项目是一个集外卖订餐手机端与后台管理系统于一体的综合性服务平台。

## 项目概述


## 技术栈

- 后端框架：Spring Boot
- 数据持久层：MyBatis-Plus
- 缓存技术：Redis
- 前端技术：HTML

## 系统角色

### 管理员

- 用户管理
- 菜品管理
- 分类管理
- 订单管理

### 用户

- 点餐
- 下单支付

## 运行环境

- JDK 1.8
- IntelliJ IDEA
- MySQL

## 部署步骤

1. 准备运行环境：安装 JDK 1.8，MySQL，并配置相应的环境变量。
2. 克隆项目代码到本地。
3. 导入数据库脚本，创建数据库及表结构。
4. 在开发工具 IDEA 中打开项目，并配置相应的数据库连接。
5. 运行项目。
